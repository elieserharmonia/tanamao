## 📱 Sobre o Projeto

O **TÁNAMÃO** é um app de divulgação de profissionais e serviços, similar à OLX, focado em conectar prestadores de serviços com clientes na região. Possui seção de Utilidade Pública com contatos de emergência e serviços da cidade.

🔗 **Demo ao vivo:** [tanamaoprofissionais.vercel.app](https://tanamaoprofissionais.vercel.app/)

## 🚀 Deploy no Vercel

Este projeto está configurado para deploy automático no Vercel.

### Como publicar:
1. Faça push para o GitHub
2. Conecte o repositório no [vercel.com](https://vercel.com)
3. O Vercel detecta automaticamente como **Static Site**
4. Deploy feito! ✅
